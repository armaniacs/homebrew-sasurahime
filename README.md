# homebrew-sasurahime

A custom [Homebrew tap](https://docs.brew.sh/Taps) for [sasurahime](https://github.com/armaniacs/sasurahime), a macOS developer cache cleaner.

## Install

```bash
brew tap armaniacs/sasurahime
brew install sasurahime
```

## Update

```bash
brew update
brew upgrade sasurahime
```

## Uninstall

```bash
brew uninstall sasurahime
brew untap armaniacs/sasurahime
```
